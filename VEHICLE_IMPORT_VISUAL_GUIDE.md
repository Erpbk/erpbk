## ğŸ“ Training Checklist

For new users learning the import feature:

- [ ] **Step 1**: Locate the import button
  - Go to Vehicles page
  - Find "Add Vehicle" dropdown
  - Click "Import Vehicles"

- [ ] **Step 2**: Download template
  - Click "Download Sample Template"
  - Open the Excel file
  - Review the sample data

- [ ] **Step 3**: Understand columns
  - Required: `plate` only
  - Optional: 22 other fields
  - Date format: YYYY-MM-DD
  - Status: 1 or 0

- [ ] **Step 4**: Prepare test data
  - Use 3-5 test vehicles
  - Fill in required plate numbers
  - Add some optional fields
  - Save the file

- [ ] **Step 5**: Upload and import
  - Select your test file
  - Choose payment account
  - Click "Import Bikes"
  - Wait for completion

- [ ] **Step 6**: Review results
  - Check success message
  - Verify vehicles in list
  - Confirm data is correct

- [ ] **Step 7**: Test error handling
  - Create file with missing plate
  - Upload and see error message
  - Fix the error
  - Re-import successfully

- [ ] **Step 8**: Test update feature
  - Export existing vehicles
  - Edit some data
  - Re-import with same plates
  - Verify updates applied
